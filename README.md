# Grovia – Intelligence That Sustains Trust  

Grovia is a full-stack AI + Blockchain carbon credit verification system combining:

- ✔️ Smart contracts (Hardhat + Solidity)
- ✔️ ML anomaly detection (FastAPI + Python)
- ✔️ Token minting logic (ERC20 carbon token)
- ✔️ Backend API (Node.js + Express + MongoDB)
- ✔️ Futuristic dark 3D animated UI (React + Vite)
- ✔️ User Dashboard + Admin Dashboard
- ✔️ Fraud detection + CO₂ prediction pipeline
